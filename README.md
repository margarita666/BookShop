# Internship project

### Project description

Online store of a printing edition (books, magazines, newspapers), with a personal account for users and the administrator.

## The project used

* Node.js
* npm
* Nest.js
* Mongo.BD



```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
